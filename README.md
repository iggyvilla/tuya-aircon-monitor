# Tuya Aircon Monitor

Track Aircon usage, and final fair energy bill split with multiple users via the Tuya KWS-302WF smart breaker. Metric tracking via Prometheus, and it is meant to be used alongside Grafana for very pretty graphs.

![Image](https://i.imgur.com/AqQee6w.jpeg)

## Discord Bot
`ac-bot.py` is a Discord bot to allow people to interface with the breaker easily, and for people to see notifications on when someone turns the breaker on/off. See images below.
